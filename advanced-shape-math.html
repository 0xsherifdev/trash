01 — Chained Boolean Operations

Stacking Operations

Real icons aren't built with a single boolean. They chain 3–5 operations in sequence. The order matters — changing the sequence gives a completely different result.

01

Build a Settings Gear Icon

Medium
Chain UNION→ UNION→ INTERSECT→ SUBTRACT

Primitives used: 1 outer circle, 1 inner circle, 4 pairs of crossed rectangles at 22.5° intervals.

Step through each phase to see how a complex icon is built entirely from basic shapes and 4 chained boolean operations.

Key insight: The order is critical. Union the teeth first, then intersect to clip, then subtract the hole. If you subtract first, you lose your cutting shape.

02

Build an Eye / Visibility Icon

Medium
Chain INTERSECT→ SUBTRACT→ UNION

Primitives used: Only circles. Two large overlapping circles create the almond eye shape (called a vesica piscis in geometry).

The intersect of two offset circles is one of the most useful compound shapes in design. It creates organic, eye-like, leaf-like, and petal-like forms from pure geometry.

Mind-blower: The Mastercard logo, the NFL logo, and countless leaf/petal icons all start from this same intersect trick.

03

Build a Shield Badge with Checkmark

Hard
Chain UNION→ INTERSECT→ SUBTRACT→ SUBTRACT→ UNION

5-step chain. The shield shape itself is a union of a rectangle + circle, intersected with a bounding box to crop the bottom. The border effect is created by subtracting a smaller copy from the larger shape (Offset Path trick).

Key technique: Offset Path subtraction. Duplicate a shape, scale it down uniformly, subtract it from the original. This creates perfect uniform borders and outlines that stroke-width can't achieve on complex shapes.

02 — Winding Rules (Compound Paths)

The Fill Rule Nobody Explains

When you put a shape inside another shape using a compound path, the inner shape can either be filled or become a hole. Winding rules control which happens — and understanding them is key to building clean logos.

Even-Odd Rule

Counts how many path boundaries a ray crosses. Odd = filled, Even = empty. Inner shapes always become holes regardless of path direction.

fill-rule: evenodd
USE

This is what Figma uses by default. It's predictable — any enclosed shape becomes a cutout. Great for letters like O, A, D, B.

Non-Zero Rule

Considers path direction (clockwise vs counter-clockwise). Same direction = filled. Opposite direction = hole. You control the result by reversing paths.

fill-rule: nonzero
USE

Illustrator's default. Reverse a path's direction (Path → Reverse) to toggle between hole and fill. Essential for compound path logos.

WHY THIS MATTERS

Real scenario: You paste an SVG logo into Figma and the counter-spaces fill in solid instead of being holes. That's a winding rule mismatch. Fix it by selecting the inner path and choosing "Reverse Path Direction" or switching the fill rule.

03 — Advanced Pathfinder Operations

Beyond the Big Four

Illustrator's Pathfinder panel has 6 additional operations that most designers never touch. Here's what they do and when they're indispensable.

Divide

Cuts all overlapping shapes into separate individual pieces along every intersection. Like using a knife at every overlap. Each fragment becomes its own shape you can color, move, or delete independently.

DIVIDE

Use case: Sticker-style illustrations, color separation, creating puzzle-piece effects. Draw your shapes loosely, divide, then color each region.

Trim

Removes the hidden portions of overlapping shapes. What you see is what you get — hidden areas behind other shapes are deleted. Shapes keep their original colors but lose their hidden parts.

TRIM

Use case: Preparing artwork for print where overlapping fills cause ink problems. Also great for simplifying complex layered illustrations.

Crop

The top shape acts as a cookie cutter window. Everything outside the top shape is removed. Similar to a clipping mask but destructive — the result is actual paths, not a masked group.

CROP

Use case: Cutting artwork to fit specific dimensions, creating viewport effects, trimming illustrations to exact bleed boundaries.

Outline

Divides shapes at intersections AND splits strokes into filled segments. Every line segment between two intersection points becomes a separate object. Removes all fills, keeps only the skeleton.

OUTLINE

Use case: Creating stained-glass effects, wireframe art, and preparing artwork for laser cutting where you need individual path segments.

Minus Back

The reverse of Subtract (Minus Front). The back shape cuts from the front shape. Stack order matters — this removes the bottom shape's area from the top shape.

MINUS BACK

Use case: When your cutting shape is behind the target. Saves you from re-ordering layers. Also useful when batch-processing multiple subtractions.

Merge

Like Unite/Union but only merges same-colored shapes. Different colors stay separate. It also removes hidden areas like Trim. Best of both worlds.

MERGE

Use case: Cleaning up complex multi-colored illustrations. Apply Merge to simplify the shape structure while preserving your color separations.

FIGMA GAP

Figma doesn't have Divide, Trim, Crop, Outline, Minus Back, or Merge. This is why Illustrator is still superior for production illustration work. In Figma, you'd need to manually simulate these with combinations of the basic four operations + Flatten. Figma's new Shape Builder tool closes the gap somewhat, but for precise control, Illustrator's Pathfinder is unmatched.

Compound Path Tricks

These techniques combine everything above into professional workflows.

Boolean → Outline Stroke → Boolean

Create a shape with booleans. Convert its stroke to a fill (Outline Stroke). Now boolean that fill with another shape. This is how you create custom letter-cut logos where type is integrated into geometric shapes.

UNION→ OUTLINE STROKE→ SUBTRACT

Expand Appearance → Divide → Recolor

Apply effects (like rounded corners, transforms, warps) then Expand Appearance to bake them in. Divide to separate regions. Now you can individually recolor each section — this is how flat illustration styles are built.

EXPAND→ DIVIDE→ RECOLOR

Offset Path Stack

Duplicate a shape 3–4 times at increasing Offset Path values. Each layer is slightly larger. Use Subtract between each pair to create concentric outline rings. Stack with different colors for the retro badge/stamp aesthetic.

OFFSET ×3→ SUBTRACT→ SUBTRACT→ COLOR

Live Corners + Boolean = Squircle

Illustrator's Live Corners (direct select → drag corner widgets) + boolean operations let you create iOS-style squircles and superellipses. Intersect a rounded square with a circle to fine-tune the corner smoothness beyond what border-radius gives you.

LIVE CORNERS→ INTERSECT→ SMOOTH
